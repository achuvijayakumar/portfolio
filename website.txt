Instead of using heavy animation libraries like GSAP, the site uses a lightweight, native approach:

Scroll Animations: Utilizes the Intersection Observer API for high-performance scroll detection. Elements with the class .animate-on-scroll trigger CSS transitions (opacity/transform) only when they enter the viewport.
Parallax Hero: A custom window.addEventListener('scroll') script calculates vertical scroll capability to translate and fade the hero text, creating depth between foreground text and background image.
Text Reveal: A "curtain up" effect is implemented using CSS masking. Text is placed inside an overflow: hidden container and translated up from 110% to 0% using cubic-bezier easing for a cinematic feel.
B. Visual Effects & UI Patterns
3D Magnetic Tilt: The "Signature Collections" cards use a custom JavaScript physics calculation.
Logic: It tracks the mouse position relative to the card's center (mousemove) and dynamically updates the CSS transform: rotateX(...) rotateY(...). This creates a hyper-realistic "physical" response without WebGL.
Luxury Cursor: A custom dual-cursor system (dot + trailing circle) replaces the default pointer.
Implementation: JavaScript tracks coordinates. The trailing circle has a programmed delay (setTimeout or CSS transition lag) to create a "fluid" drag sensation.
Atmospheric Particles: A lightweight particle system is built in pure JS (
createParticles
